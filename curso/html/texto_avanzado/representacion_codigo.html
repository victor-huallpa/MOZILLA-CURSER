Representacion de Codigo Informatico

Existe una serie de etiquetas HTML para representar codgio Informatico

tiquetas

  • <code> para marcar fragmetos de codigo
  • <pre> es para representar espacios en blanco identicos al de tu codigo
  • <var> para marcar nombres de variables
  • <kbd> para marcar entradas de teclado
  • <samp> para marcar lasida de un programa de ordenador

Ejemplo

var para = document.querySelector('p');

    para.onclick = function() {
      alert('¡Guau!, ¡deja de apretar!');
    }

No debes utilizar elementos de presentación como <font> y <center>.
