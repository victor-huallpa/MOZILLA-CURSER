otros atrubutos que puedes usar
width y height: esto ayuda a redimencionar el video, pero esto solo mantiene sus propiedades, cuado lo haces lo hace escalable tambien se puede hacer con css.
autoplay: este atributo hace que el video se reprodusca automaticamenete una ves argado el archivo.
loop: hace que el video o audo se reprodusca de nuevo cada que finalice
muted: hace que los multimedia se reproduscan con el sonido apagado
poster: en este atributo puedes incrustar una imagen antes de reproducir un video
preload: su utiliza para almacenar en un boffer archivos grandes, contiene valores, none, auto y metadata
