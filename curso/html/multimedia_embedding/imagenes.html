IMAGGENES EN HTML


Al principio de la web todod era puro texto, pero con el tiempo combio rapidamente, ya que ahora puede ser mas dinamica y puede incluir imagenes y demas

Elementos para incrustar iamgenes

<img>:
esta etiqueta es comun para incrutar imagenes en sitios web
Al igual que los vinculos estas tienen que especificarse la ruta de la imagen para poder omstrarla ya sea que este en el servidor o en la nuve, en caso este en el servidor estas rutasp ueden ser realtivas o absolutas , y si estan en la nuve solo se requiere la url de la imagen. esta ruta se coloca dentro de un atributo como ya se menciono es aprecido a los enlaces, pero no es el atributo href para este etiqueta se usa el atributo src
otros atributos: width, height, tittle, los dos primeros tributos ayudan con las dimenciones de la imagen pero se recomienda hacerlo con css, en cuanto a ltittle ayuda con el tooltip de la descripcion de la imagen a l pasar el cursor
