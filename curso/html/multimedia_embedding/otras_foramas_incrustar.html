INCRUSTACION MEDIANTE IFRAME, OBJECT Y EMBED


al principio estas fueron las formas de incrustar contenido a las paginas web pero no fueron las mejores opciones aunque iframe aun se sige usando para incrustra contenido de otros sitios aun que es riesgoso.


  • <iframe>
  • <object>
  • <embed>

iframe: esta etqiueta se usa para incrustar contenido de otras paginas ya sean audio visuales o solo visuales
con esto puedes incruatr videos, mapas, imagenes, paginas web externas, entre otras.

Strubutos
allowfullscreen: puede colocar el iframe en pantalla completa, pero lo peudes hacer tambein con js o alguna api
frameborder: este atributo se puede colocar en 1 o 0, ayuda con la colocacion de un borde, segun sea necesario se recomienda ahcerlo con css
src: aqui es donde se coloca la url de contenido que deseas incrustar.
wifth y height: esto determina las dimenciones del iframe, tambein se recomienda realizarlo con css
sandbox: ayuda con la mayor configuracion de seguridad en la incrustacoin de un elemento.

nota: el uso de iframe solo se recomienda cuando laruta contenga HTTPS
tener cuidado con los derechos de autos
