ENVOLTURAS NO SEMANTICAS

Hay ocaciones en las que las envolturas semanticas ya no existen para nuestro trabajo, es donde se llegan usar estas envolturas.

Estas envoltursa son:

<div>
esta etiqueta se utiliza para agrupar conteido y poerdarle un atrubuto de clase y/o id
tambien se utiliza para estilar css o dar alguna funcionalidad con js
<span>
se utiliza cuando no se nos ocurre ningun otro elemento semantico de textoen el que incluir contenido

NOTA: Si en caso no quieres usar la etiqueta <aside> se puede usar la etiqueta <section>
