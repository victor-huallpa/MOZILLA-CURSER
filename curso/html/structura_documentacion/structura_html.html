ESTRUCTURA DE UNA PAGINA WEB

Normalmente una pagina web cuenta con:

Encabezado
Es la parte superior donde se coloca la frase, logo y detalles importantes de una pagina web
Esta representado por la etiqueta <header>
Menu de Navegacion
Es la parte donde se coloca el contenido de la pagina web Los enlaces que vinculan a toda la informacion de la pagina web
Esta representado por la etiqueta <nav>
Contenido Principal
Ocupa casi toda la parte de la pagina web y es donde se centra todo el contenido de la pagina web Informaicon
Esta representado por la etiqueta <main>
Barra Lateral
Esta se ubica en la parte lateral de la pagina normalmente se coloca linsk de redireccion a mas conteido de la pagina web y se encuentra dentro de la etiqueta <main>
Esta representado por la etiqueta <aside>
Pie de Pagina
Es la arte inferior de la pagina web, en esta parte se coloca informacion para hacer SEO
Esta representado por la etiqueta <footer>

otras etiquetas

<section>
se usa para agrupar cada parte de una pagina web
<article>
encuadra un bloque de contenido que tiene sentido por si mismo
