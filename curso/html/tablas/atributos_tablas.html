Atrib utos avanzados de tablas

Estos atributos ayudan con la agrupacion de columnas o filas y dar una mayor precision a los
lectores de pantalla y el usuario sepa como van relacionada toda la informacion

scope
se aniade al elemento <th> para indicar a los lectores de pantalla a quie nva dirigido el encabezado
Ejemplo
Compra Ubicación Fecha Revisión Coste (€)
Corte de pelo Peluquería 12/09 Gran idea 30
este atributo tiene valores, col, row, colgroup y rowgroup, estos dos ultimos agrupan encabezados y subencabezados
id y headers
con id se aniade ides unicos para cada <th> y posteriormente vincular a los atributos headers que van dentro de los elementos <dt>
Esto ayuda de manera mas eficiente que scope al agrupar filas o columnas de una tabla
Ejemplo
Compra Ubicación Fecha Revisión Coste (€)
Corte de pelo Peluquería 12/09 Gran idea 30
